REST API Endpoints

جميع الـ endpoints متاحة الآن للتجربة

GET / هذه الصفحة (Dashboard)
GET /api/health فحص صحة النظام + حالة DB
GET /api/info معلومات عن التطبيق
GET /api/version رقم الإصدار
GET /api/metrics Prometheus metrics
GET /api/users?page=1&per_page=20 قائمة المستخدمين (مع pagination)
POST /api/users إنشاء مستخدم جديد
GET /api/users/:id عرض مستخدم
PATCH /api/users/:id تعديل مستخدم
DELETE /api/users/:id حذف مستخدم
GET /api/users/count عدد المستخدمين الكلي

كيف تبدو تطبيقات NovaX

// apps/novax-app/src/main.rs use novax::prelude::*; #[novax::main] async fn main() { let app = App::new() .with_route("GET", "/", || async { "Hello, NovaX!" }); app.serve("0.0.0.0:3000").await.unwrap(); } // #[novax::route] يتولّى: التحقق من النوع، توليد OpenAPI، // تسجيل المسار، إعداد الـ tracing، وكل ما يلزم تلقائيًا #[novax::route(GET", "/users/{id}")] pub async fn get_user( _ctx: Context, id: Path<Uuid>, ) -> Result<Json<User>, ApiError> { let user = Users::find_by_id(&id).await?; Ok(Json(user)) }
